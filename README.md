# 🏡 California House Price Prediction using Machine Learning

A modern Machine Learning web application that predicts California house prices based on housing characteristics such as location, median income, population, number of rooms, and other important features.

The application is developed using **Python**, **Flask**, **Scikit-learn**, **Pandas**, **NumPy**, **Matplotlib**, **HTML**, and **CSS**.

---

## 🚀 Project Overview

This project uses a **Random Forest Regressor** trained on the California Housing Dataset to estimate house prices.

The web application allows users to enter housing information through an interactive interface and instantly predicts the estimated house price.

It also includes a modern analytics dashboard with visualizations to help understand the dataset and model.
